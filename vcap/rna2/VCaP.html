Visualizations

MA-plot

log2 fold changes: log(experimental) - log(control)

Points will be colored red if the adjusted p value is less than 0.1. Points which fall out of the window are plotted as open triangles pointing either up or down.

It is more useful visualize the MA-plot for the shrunken log2 fold changes, which remove the noise associated with log2 fold changes from low count genes without requiring arbitrary filtering thresholds.

Count Plot by Gene

It can also be useful to examine the counts of reads for a single gene across the groups. A simple function for making this plot is plotCounts, which normalizes counts by sequencing depth and adds a pseudocount of 1/2 to allow for log scale plotting. Here, the plot for the lowest p value gene is given.

Exporting Significant Results

First, order our p-values by the smallest.

resOrdered <- res[order(res$pvalue),]

Now, filter out by p-value threshold and log2fc and export it as CSV.

resSig <- subset(resOrdered, padj < 0.05)
resSig <- subset(resSig, abs(log2FoldChange) > 1)
write.csv(as.data.frame(resSig), file="condition_ERG_results.csv")
